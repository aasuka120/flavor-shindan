# フレーバー診断

> あなたを“味”にたとえると？ — 20問・5段階回答・約2分で16タイプの「フレーバー」を診断するZ世代向けバイラル診断サイト。
> 戦略は [STRATEGY.md](./STRATEGY.md)、調査・アイデアの根拠は [RESEARCH_AND_IDEAS.md](./RESEARCH_AND_IDEAS.md) を参照。

## 動かし方

ビルド不要の静的サイト。どちらでもOK:

- `index.html` をダブルクリックしてブラウザで開く
- ローカルサーバで開く(推奨):
  ```
  python -m http.server 8000
  # → http://localhost:8000
  ```

デプロイは Cloudflare Pages にフォルダごと置くだけで動く(設定不要)。

## ファイル構成

```
index.html        画面の骨格(ランディング/クイズ/調合中/結果)
css/style.css     デザイン一式(レトロ駄菓子パッケージ風デザインシステム)
js/data.js        ★コンテンツの本体。質問・16タイプの全コピー・SVGキャラ・相性
js/app.js         画面遷移・クイズ進行・結果ページ描画・シェア導線
js/card.js        保存用カード画像(1080×1920 PNG)のCanvas生成
```

## コンテンツを編集したいとき

**すべて `js/data.js` だけで完結する。**

- 質問を変える → `QUESTIONS`(各問は4軸のどれかに属する。`axis` と回答の軸文字を対応させること)
- タイプのコピーを磨く → `TYPES.{id}` の `desc / manual / aruaru / love / friend / bestReason / worstRoast` など
- キャラの見た目 → `TYPES.{id}.svg`(viewBox 200×200、線色 #3a2c23・線幅5で統一)
- レア度% → `rarity`(16タイプ合計が100になるよう調整)

## 診断ロジック

- 4軸 × 各5問 = 20問。一人称の文に5段階(そう+2 / たぶんそう+1 / わからない0 / たぶんちがう-1 / ちがう-2)で回答
- 各軸に逆方向の質問を2問混ぜてあり(`dir:'b'`)、採点時に符号反転(回答パターン化の防止)
- 軸ごとに-10〜+10点。プラスなら前者・マイナスなら後者。0点ちょうどの同点は「最初に0以外を答えた質問」の向きで決める
- 配合% = (得点+10)/20×100。結果ページの「あなたの配合」ゲージに表示される
- 軸: 濃さ[こってり/あっさり] 温度[ホット/アイス] 後味[甘口/辛口] 入手性[定番/限定] → 2^4 = 16タイプ
- 相性: 濃さ逆・温度同・甘辛逆・入手性逆 が好相性(`pairScore`)。ベスト/ワーストは全タイプで相互ペアになっている

## 設計メモ

- 登録なし・個人情報ゼロ。自分の結果は localStorage、結果ページは `?t={typeId}` で共有可能
- 結果カードは画面表示用(DOM)と保存用(Canvas PNG 9:16)の2系統。デザインは同一
- レア度は現状プリセット値。バックエンド(Cloudflare Workers + KV)を足せば実カウントに置き換え可能
